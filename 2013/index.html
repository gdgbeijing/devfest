张文杰

Intel 资深架构师

WebRTC 技术的实现及拓展应用

13:15 - 14:05

WebRTC 是 W3C 制定的一套 HTML5 实时音视频通信技术标准。本主题覆盖 WebRTC 的技术起源、发展、现状及相关的技术实现细节，同时介绍 Intel 基于 WebRTC 技术的拓展应用服务平台。

关于张文杰

Intel亚太研发有限公司资深构架师， 2003年加入Intel从事网络编译器测试优化、项目创新、Web技术优化及应用平台开发工作。有十年以上的软件开发与测试、系统优化、本地项目孵化及创新的经验。

Alpha Yang

Google Apps Consultant

Google App Engine & Cloud Overview

14:05 - 15:00

介绍 Google App Engine 的最新特性，涵盖整个 Google 云平台的介绍，让你站在云计算的最前沿，一揽众山小。

关于 Alpha

Alpha 在 Google Apps 和 Google Cloud Platform 有超过5年的实战经验。在帮助客户使用最新的云端技术提高生产效率的同时，降低投入成本，获得更高的技术投入回报。

Google Dart GDE

Web 的未来 - 新技术和新语言

关于韩国恺

韩国恺，高级工程师、技术经理，多年开发和管理经验。擅长 Web 前后端技术、网站架构和运维管理，熟悉多种编程语言。做过企业应用和互联网服务等不同类型产品。国内的第一位 Google GDE，北京 GDG 组织者。

李高禄(unbug)

网易前端开发攻城师,MIHTool作者.

Polymer Code Lab

16:05 - 17:00

Polymer 是一个全新的基于 Web Components 基础之上的 Web 库，为 Web 平台上浏览器的演进提供支持。本课程为 Code Lab 形式，即根据准备好的教程动手学习 Polymer 库。听众需自带笔记本。

由于会场网络条件有限，请参加 Polymer Code Lab 的参会者做如下准备：
1.需要python2.7+环境(如何安装python)或者GAE(如何安装Google App Engine),建议两者都部署好.
2.需要nodeJS环境(如何安装nodeJS)和bower(npm install -g bower)
3.需要Polymer project 源码包, 最新版下载
4.需要示例项目TodoMVC, https://github.com/Polymer/todomvc
5.当然Chrome是必须的

关于李高禄

网易前端开发攻城师,MIHTool作者.目前专注移动前端. Twitter: @unbug 微博: @听奏

周爱民(aimingoo)

豌豆荚架构师

smali 汇编语言的设计与实现

13:15 - 14:05

smali 是 .dex 文件反编译生成的汇编代码。也可以直接手写 smali 代码。他是目前唯一能完整实现反编译和再编译的、面向 Dalvik 上的汇编了。您将了解到 smali 与 .dex 和 Dalvik 的关系，以及（最关键的）这个汇编语言的设计与实现。

关于周爱民

周爱民(Aimingoo)，十余年的软件开发、项目管理、团队建设的经验。前支付宝业务架构师，前盛大网络平台架构师，现任豌豆荚架构师。著有《Delphi源代码分析》、《JavaScript语言精髓与编程实践》、《大道至简》、《大道至易》等专著。目前主要从事软件工程、体系架构和语言基础方面的研究与实践。

倪超(木鱼)

金山网络

Chrome扩展开发(猎豹浏览器12306订票助手)

15:15 - 16:05

去年春运期间，一款浏览器的订票助手备受大众关注，一不小心还拖垮了 Github 。12306订票网站助手是使用是 Chrome 和 Firefox 上提供的扩展机制。由浏览器提供脚本扩展入口，将自身的脚本注入到12306订票网站内，从而完成定制界面和功能的使命。

关于倪超

倪超（网名木鱼）互联网从业六年，长期从事互联网应用以及企业ERP应用领域开发。07 至13年就职于一商业地产企业主持ERP系统中自有系统的开发建设，目前加入金山网络。

叶晨光

北京帕罗奥图科技有限公司创始人&CEO

以Google Class为代表的穿戴式未来

16:05 - 16:50

谷歌X LAB多年前开始研究人类继手机终端之后的新的高科未来，通过在人本身介质上加强和五官的直接接触。以谷歌眼镜为代表的穿戴式和头戴式智能设备，更方便的人类通讯、更易完成人机交互、加深人与人本体之间的信息交流。目前谷歌眼镜名为拓展现实虚拟成像投射眼镜，以协助人类更好的识别周边环境、促进更多信息互动，通过智能眼镜为终端完成未来的物联网链接，使人们生活在足够信息量的环境当中。

关于叶晨光

通信、传媒行业背景，穿戴式设备研究者，移动互联网分析师。从业并服务过诺基亚、谷歌youtube等通信和互联网公司，目前初创穿戴式研发公司AltoTech，对未来穿戴设备特别是头戴式做研究，包括人脸识别、AR现实增强等技术。

赵宇

梆梆安全 VP

安卓应用的安全保护

16:50 - 17:25

随着移动互联网的爆发式增大，移动应用市场也越来越火爆，在移动应用市场繁荣的背后，安全问题也日趋严重， 目前市场上60%以上的APP都是被篡改过的，很多开发者开发的应用被二次打包或被塞进很多广告，如何从技术上保护开发者APP的安全？是目前很多开发者面临的问题。

关于赵宇

梆梆安全联合创始人&副总裁。中国科学院硕士，有超过8年的增值业务提供商的运营和管理工作经验，曾就职于 万向通信、保利通信等公司。
